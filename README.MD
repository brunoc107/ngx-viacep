# brunoc-ngx-viacep

Um módulo para buscar endereço e CEP.

## Instalação

Para instalar, utilize o npm:

```bash
$ npm install @brunoc/ngx-viacep --save
```

Ou o Yarn:

```bash
$ yarn install @brunoc/ngx-viacep --save
```
